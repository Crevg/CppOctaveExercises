#############################################################################
Ubicarse en la carpeta del Ejercicio 3 antes de realizar los pasos siguientes

#############################################################################
Para compilar:
* Ejecutar el archivo CompileScript

Para correr:
*Ejecutar el archivo RunScript con los argumentos escritos de inmediato,
en el siguiente orden:
Argumento 1: a (magnitud de x al cuadrado).
Argumento 2: b (magnitud de x).
Argumento 3: c (magnitud constante).

Ejemplo con a = 1 b = 3 c = -4:
$ ./CompileScript
$ ./RunScript 1 3 -4

#############################################################################
NOTA: Se deben tener permisos de escritura en el directorio ya que se creará
una carpeta llama "bin" donde se estará ubicado el archivo objeto (output).
